# Lime Basket DS

Lime Basket is a design system built on react.

## Installation

Use the package manager [npm](https://npm.com) to install lime-basket-ds.

```bash
npm install lime-basket-ds
```

## Usage

```react
import { Button } from "lime-basket-ds";

const App = () => (<Button />);
```
