# Foobar

Lime Basket is a design system built on react.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install lime-basket-ds.

```bash
pip install lime-basket-ds
```

## Usage

```react
import { Button } from "lime-basket-ds";

const App = () => (<Button />);
```
